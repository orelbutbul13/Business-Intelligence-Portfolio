# Data Analytics & Business Intelligence Portfolio

Collection of Business Intelligence and data analytics projects focused on transforming raw data into structured, decision-ready insights.

These projects demonstrate SQL-based analysis, dashboard development, KPI modeling, return-risk diagnostics, and performance evaluation across investment, retail, and e-commerce domains.

---

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Venture Capital Investment Analysis](https://github.com/orelbutbul13/Business-Intelligence-Portfolio/tree/main/Venture-Capital-SQL-Analysis) | SQL-based analysis of venture capital funding activity evaluating startup performance, investment patterns, acquisition trends, and market concentration. |
| 2 | [Superstore Profitability Analysis](https://github.com/orelbutbul13/Business-Intelligence-Portfolio/tree/main/Retail-Profitability-Tableau-Dashboard) | Tableau dashboard analyzing retail profitability, margin drivers, return behavior, and geographic performance trends. |

| 4 | [Retail Return Intelligence System](https://github.com/orelbutbul13/Business-Intelligence-Portfolio/tree/main/Retail-Returns-Analysis-Tableau) | KPI-based return analysis and Tableau dashboard identifying category risk concentration, geographic return patterns, seasonal volatility, and customer return behavior. |
| 5 | [Shopify Marketplace Analysis](https://github.com/orelbutbul13/Business-Intelligence-Portfolio/tree/main/Shopify-Marketlace-Analysis-Power-Bi) | Power BI project analyzing marketplace scale, developer responsiveness, and community-weighted sentiment using advanced DAX modeling. |

## Core Competencies

- **SQL:** Query design, relational data analysis, and schema architecture.
- **Power BI & Tableau:** Dashboard architecture, DAX modeling, and advanced visualization.
- **Business Intelligence:** KPI development, performance benchmarking, and trend evaluation.
- **Advanced Analytics:** Return rate normalization, risk diagnostics, and multi-dimensional segmentation.
- **Marketing Analytics:** Funnel analysis and cohort retention modeling.
- **Communication:** Executive-level reporting and strategic insight communication.
