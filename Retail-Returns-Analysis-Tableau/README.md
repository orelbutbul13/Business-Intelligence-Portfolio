# Retail Return Intelligence System

Business Intelligence and data visualization project analyzing retail return behavior using Tableau to identify root causes of elevated return rates, operational risk concentration, and return-driven margin exposure.

## Business Objective

Retail leadership requires structured visibility into return behavior impacting operational efficiency, customer experience, and profitability.

This project transforms transactional retail data into actionable business intelligence supporting:

- Return rate normalization and KPI modeling  
- Category-level return risk analysis  
- Sales vs. returns correlation assessment  
- Geographic return concentration evaluation  
- Seasonal volatility analysis  
- Customer return behavior segmentation  

The objective was to convert raw operational data into executive-ready insights through interactive dashboard modeling and return-risk diagnostics.

---

## Dataset Overview

The analysis is based on the Superstore dataset, including:

- Orders  
- Customers  
- Products  
- Categories and sub-categories  
- Sales metrics  
- Geographic information  
- Returns data  

Returns were integrated using a left-join architecture and transformed into a deterministic return-rate KPI.

---

## Analytical Focus Areas

- Return rate by category and sub-category  
- Sales vs. returns correlation modeling  
- Geographic return concentration analysis  
- Monthly and seasonal return volatility  
- Repeat-customer return behavior  
- Composite cross-dimensional risk evaluation

## Tools and Techniques

- Tableau

---

## Key Findings

- Technology exhibits the highest return rate (~27%) and represents the largest operational return risk concentration.  
- Returns are positively correlated with sales volume, but certain sub-categories show disproportionate return behavior.  
- Geographic return concentration is elevated in Western and Southwestern states.  
- Return rates fluctuate seasonally, indicating promotional or demand-cycle impact.  
- A subset of repeat customers contributes disproportionately to total return volume.
